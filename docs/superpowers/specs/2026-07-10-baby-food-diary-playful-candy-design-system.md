# 宝宝辅食日记｜糖果乐园设计系统决策

已批准采用“糖果乐园”儿童感方向。全局设计规范位于 [`design-system/泡泡的辅食日记/MASTER.md`](../../../design-system/泡泡的辅食日记/MASTER.md)。

该系统保留珊瑚橙作为主操作色，将粉紫、薄荷绿和奶油黄限定为辅助表面与插画背景；因此既有儿童感，也不会削弱“记录一餐”“安全复吃”“过敏/观察中”等业务信息的可读性。

Master 已覆盖颜色、字体、间距、布局、导航、按钮、卡片、标签、表单、状态、动效、无障碍和五个页面的应用边界。后续页面设计或代码实现应优先读取该规范；没有页面级 override 时不得偏离其中的 Token 与组件规则。
