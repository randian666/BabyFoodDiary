# 设置页与 JSON 导出设计说明

## 目标

为“泡泡的辅食日记”增加独立设置页：可编辑宝宝资料、展示头像，并将当前示例数据导出为 JSON 后通过系统分享面板发送或保存。

## 入口与导航

- 首页右上角的通知图标替换为设置齿轮。
- 点击后以全屏 Sheet 展示设置页；左上角返回，关闭后回到首页。

## 设置页

- 顶部展示圆形宝宝头像与“更换头像”按钮；本期用系统图标与渐变头像占位，不接照片选择器。
- 信息区包含昵称“泡泡”、月龄“10个月20天”、性别“男宝”。字段使用本地 `@State`，可在页面内编辑。
- 数据管理区提供“导出 JSON”主操作，解释导出范围为宝宝信息、菜谱与餐次记录。

## JSON 导出

- 以 `Codable` 结构生成 UTF-8 JSON 文件；包含 `baby`、`recipes` 和 `mealRecords` 三个顶级字段。
- 文件保存至应用临时目录，名称为 `baby-food-diary-export.json`。
- 使用 iOS 系统分享面板分享该文件；不接网络、不上传数据、不持久化。

## 视觉与验证

- 沿用暖杏橘背景、22pt 卡片圆角、暖橘软阴影、44pt 以上操作区域。
- 使用 Xcode 设备目标构建验证编译。
